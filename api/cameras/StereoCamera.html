Properties

[property:Float aspect]

Default is *1*.

[property:Float eyeSep]

Default is *0.064*.

[property:PerspectiveCamera cameraL]

左侧相机.这将添加到[page:Layers layer 1]第1层 - 左侧摄像头要渲染的对象也必须添加到此层.

[property:PerspectiveCamera cameraR]

右侧相机.它被添加到[page:Layers layer 2](第2层) - 右边相机要渲染的对象也必须添加到该图层.
