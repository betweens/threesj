相机

相机的抽象基类. 构建新摄像机时,应始终继承此类.

构造函数

[name]()

创建一个新相机.请注意,此类不应直接调用; 你可能想要一个PerspectiveCamera或OrthographicCamera.

属性

有关常见属性,请参阅基础[page:Object3D]类

[property:Boolean isCamera]

用于检查此类或派生类是否为摄像机. 默认为true.

您不应该更改此内容，因为渲染器在内部使用它进行优化.

[property:Layers layers]

相机所属的图层, 这是[page:Object3D]的继承属性.

在渲染相机的视点时,对象必须与相机共享至少一个图层.

[property:Matrix4 matrixWorldInverse]

这是matrixWorld的反转. MatrixWorld包含Matrix,它具有Camera的世界坐标变换.

[property:Matrix4 projectionMatrix]

这是包含投影的矩阵.

方法

有关常用方法,请参阅基础[page:Object3D]类

[method:Camera clone]()

返回具有与此相同属性的新相机.

[method:Camera copy]( [param:Camera source], [param:Boolean recursive] )

将源摄像机中的属性复制到此摄像机中.

[method:Vector3 getWorldDirection]( [param:Vector3 target] )

[page:Vector3 target] — 结果将被复制到此Vector3中.

返回表示摄像机正在查看的世界坐标空间方向的Vector3.(注意:相机向下看其局部的负z轴).
