概览

Three.js可以使用WebGL在所有现代浏览器上渲染场景. 对于较旧的浏览器, 尤其是Internet Explorer 10及更低版本,您可能必须回退到其他[link:https://github.com/mrdoob/three.js/tree/master/examples/js/renderers (renderers)]渲染器之一（CSS2DRenderer, CSS3DRenderer, SVGRenderer, CanvasRenderer）. 此外,您可能必须包含一些polyfill,特别是如果您使用[link:https://github.com/mrdoob/three.js/tree/master/examples /examples]文件夹中的文件.

提示: 如果您不需要支持这些旧浏览器，则不建议使用其他渲染器，因为它们比WebGLRenderer慢，并且支持的功能少.

支持WebGL的浏览器

Google Chrome 9+, Firefox 4+, Opera 15+, Safari 5.1+, Internet Explorer 11 and Microsoft Edge. 你可以找到哪些浏览器支持[link:https://caniuse.com/#feat=webgl browsers suppor WebGL].

JavaScript语言功能或Web API在three.js中使用

以下是three.js中使用的一些功能. 其中一些可能需要额外的polyfill.

Feature Use Scope Modules
Typed Arrays Source BufferAttribute, BufferGeometry, etc.
Web Audio API Source Audio, AudioContext, AudioListener, etc.
WebVR API Source WebVRManager, etc.
Blob Source FileLoader, etc.
Promise Examples GLTFLoader, GLTFExporter, WebVR, VREffect, etc.
Fetch Examples ImageBitmapLoader, etc.
File API Examples GLTFExporter, etc.
URL API Examples GLTFLoader, etc.
Pointer Lock API Examples PointerLockControls

Polyfills

只需根据您的要求导入polyfills. 以IE9为例, 您需要至少补充这些功能:

  • Typed Arrays
  • Blob
